# Deadlift Variace

3 variace deadliftu + technika spodní fáze pro zlepšení síly a techniky od závodního powerliftéra.

## Obsah

1. **Dotahy** – horní fáze (lock), agresivní zavírání pánví
2. **Pauzovaný Deadlift** – pauza u kolen, zlepšení přechodu mezi fázemi
3. **Deadlift s Expanderem** – technika držení osy u nohou, korekce dráhy
4. **Technika Spodní Fáze** – zapojení nohou, správná startovní pozice
